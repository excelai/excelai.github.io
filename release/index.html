版本更新历史

Release 1.8.1

发布日期：2023-12-02
更新内容：

  • 【自主对话】当中新增【增强版】选项（GPT-4.0模型）、新增联网支持（对话过程中可以自动通过互联网获取相关信息）、新增搜索支持（对话过程中会自动通过Bing搜索获取相关信息），以上功能都可以在【自主对话】的【选项】当中进行开启。
  • 【智问公式】当中增加【增强版】选项，勾选状态下使用GPT-4.0模型（没有勾选时默认采用GPT-3.5-Turbo模型）。【增强版】这个选项不在永久免费的范围内，超出15次免费试用以后就会消耗AI电量。
  • 【分享计划】当中增加商务代理的详情显示（点击邀请人数时显示）。

Release 1.8.0

发布日期：2023-09-19
更新内容：

  • 新增对WPS的支持，发布适用于WPS的插件版本（需要另外下载安装），自此进入Excel/WPS双版本运营时代
  • 【公式改错】功能现在允许对没有产生错误值的公式进行查错处理，例如，可以对=IF(2<A2<7, "是", "否")这类语法或逻辑错误的公式进行纠错
  • 修复一些BUG

  • 修复部分用户在使用【图片生成】功能时出现“下载和拆分图片时发生错误”的问题。

发布日期：2023-09-05
更新内容：

  • 修复一些BUG

Release 1.7.2

发布日期：2023-09-01
更新内容：

  • 【图片生成】新增以图生图功能（垫图），可以在图片生成界面的【设置】按钮当中选取使用
  • 【图片生成】新增快速通道的选取功能，快速通道比上一版本的极速通道生成图片的速度稍慢，但费用也会更低。当前版本支持快速通道和极速通道的自由切换，可以在图片生成界面的【设置】按钮当中进行切换
  • 【图片生成】当中原有的尺寸比例选择项改到【设置】按钮当中进行选择

Release 1.7.1

发布日期：2023-08-23
更新内容：

  • 新增【图片生成】功能，对接Midjourney模型，支持使用中文或英文的提示文字自动生成图片。每次调用可生成四张图片（每次消耗4500AI电量，前8次免费使用），可以选定图片进行下载、微调、设置为用户头像等操作
  • 【图片生成】当中还提供了丰富的提示词生成功能，可以根据分类选取需要的提示词，并对提示词进行智能优化
  • 支持用户基于【图片生成】功能自定义头像（【个人中心】中所显示的头像图片）
  • 新增AI服务的备用地址，可以自动切换，解决个别地区（例如厦门）无法正常使用AI服务的问题

  • 【智问代码】功能增加【增强版】选项，勾选后可以使用GPT-4.0模型，提高代码生成的质量

Release 1.1.3 试用版

发布日期：2023-04-12
更新内容：

  • 免费试用版内部测试
  • 智问公式：向AI解释需要实现的公式要求或目标，由AI自动给出公式
  • 公式释义（含右键菜单）：选中有公式的单元格或手动输入公式，由AI来帮您解释分析这个公式的含义
  • 公式改错（含右键菜单）：选中产生错误值的公式单元格，由AI来帮您分析错误原因，并且推荐可以替代的修正公式
  • 智换公式（含右键菜单）：根据选定的一列数据，提出转换要求或给出转换示例，由AI帮您设计一个可以向下填充的公式，完成所有数据的批量转换
  • 智换数据（含右键菜单）：根据选定的一列数据，提出转换要求或给出转换示例，由AI帮您完成所有数据的批量转换（试用版仅支持转换前20行）
  • 智能翻译（含右键菜单）：多语种翻译，并提供不同文字风格的设定
  • 数据生成
    • 时间维度：根据选定的行业、职能部门和业绩指标数量，自动生成关键指标的虚拟数据，横向颗粒度取决于用户选择的时间颗粒（试用版中业绩指标数量和行数均仅限最小数量）
    • 人员维度：根据选定的行业、职能部门和业绩指标数量，自动生成关键指标的虚拟数据，并且可以在人员字段当中添加其他字段，横向颗粒度为人员（试用版中业绩指标数量和行数均仅限最小数量）
  • 精彩问答：看看别人都有什么精彩的问答，目前仅显示智问公式功能当中用户分享的问答
  • 自主对话：聊天方式与AI直接问答，具备记忆深度设置（试用版仅限5组对话的记忆深度设置）、聊天记录导出为文件、导入聊天记录的功能
  • 试用版相关设置：
    • 首次安装时需要填写邀请码激活
    • 可以生成自己的邀请码，邀请好友来参与试用
    • 每台设备默认试用时间2天，每邀请一位好友安装使用后增加2天试用时间
