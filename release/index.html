版本更新历史


Release 1.1.3

发布日期：2023-04-12
更新内容：

  • 免费试用版内部测试
  • 智问公式：向AI解释需要实现的公式要求或目标，由AI自动给出公式
  • 公式释义（含右键菜单）：选中有公式的单元格或手动输入公式，由AI来帮您解释分析这个公式的含义
  • 公式改错（含右键菜单）：选中产生错误值的公式单元格，由AI来帮您分析错误原因，并且推荐可以替代的修正公式
  • 智换公式（含右键菜单）：根据选定的一列数据，提出转换要求或给出转换示例，由AI帮您设计一个可以向下填充的公式，完成所有数据的批量转换
  • 智换数据（含右键菜单）：根据选定的一列数据，提出转换要求或给出转换示例，由AI帮您完成所有数据的批量转换（试用版仅支持转换前20行）
  • 智能翻译（含右键菜单）：多语种翻译，并提供不同文字风格的设定
  • 数据生成
    • 数据维度：根据选定的行业、职能部门和业绩指标数量，自动生成关键指标的虚拟数据，横向颗粒度取决于用户选择的时间颗粒（试用版中业绩指标数量和行数均仅限最小数量）
    • 人员维度：根据选定的行业、职能部门和业绩指标数量，自动生成关键指标的虚拟数据，并且可以在人员字段当中添加其他字段，横向颗粒度为人员（试用版中业绩指标数量和行数均仅限最小数量）
  • 精彩问答：看看别人都有什么精彩的问答，目前仅显示智问公式功能当中用户分享的问答
  • 自主对话：聊天方式与AI直接问答，具备记忆深度设置（试用版仅限5组对话的记忆深度设置）、聊天记录导出为文件、导入聊天记录的功能
  • 试用版相关设置：
    • 首次安装时需要填写邀请码激活
    • 可以生成自己的邀请码，邀请好友来参与试用
    • 每台设备默认试用时间2天，每邀请一位好友安装使用后增加2天试用时间
